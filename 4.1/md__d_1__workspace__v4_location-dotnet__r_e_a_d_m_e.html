Introduction

This repository contains the geo-location engines for the .NET implementation of the Pipeline API.

Pre-requesites

Visual Studio 2017 or later is recommended. Although Visual Studio Code can be used for working with most of the projects.

The Pipeline engines are written in C# and target .NET Standard 2.0.3 and .NET Core 2.0.

Solutions and projects

  • FiftyOne.GeoLocation - Geo-location engines and related projects.
    • FiftyOne.GeoLocation.DigitalElement - A .NET engine which retrieves geo-location results by consuming data from a DigitalElement endpoint.
    • FiftyOne.GeoLocation.Nominatim - A .NET engine which retrieves geo-location results by consuming data from a Nominatim endpoint.
    • FiftyOne.GeoLocation.Core - Shared classes used by the geo-location engines.
    • FiftyOne.GeoLocation - Contains the geo-location engine builders.
    • FiftyOne.GeoLocation.Cloud - A .NET engine which retrieves geo-location results by consuming data from the 51Degrees cloud service.

Installation

You can either reference the projects in this repository or you can reference the NuGet packages in your project:

Install-Package FiftyOne.GeoLocation -Version 4.1.0

Make sure to select the latest version from NuGet.
