◆ DigitalElementEngine_TestRefresh()

void GeoLocation.DigitalElement.Tests.DigitalElementEngineTests.DigitalElementEngine_TestRefresh ( )

Test that the reload method does actually reload the data used by the engine.

  1. Start the engine with a city file that will return the name 'Reading' for code 13642.
  2. Process using the engine and arrange for the result to contain the city code 13642. Make sure the result for the 'Town' property is 'Reading'.
  3. Refresh the engine, causing it to load a new version of the cities file where the code 13642 now has the name 'Slough'.
  4. Process using the engine again. This time the result for the 'Town' property should be 'Slough'.
