FiftyOne.GeoLocation.DigitalElement.FlowElements.DigitalElementEngineBuilder Class Reference

Detailed Description

Fluent builder for the DigitalElementEngine.

Contains methods to set up the four decode files required to gather a meaningful result from the Digital Element Netacuity API and the license key required to access the Digital Element GeoMPrint API.
